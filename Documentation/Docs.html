Project Overview

     Welcome to mini-PS, Our project is an open source mini-PlayStation application that contains 4 open source games (Alien invasion-Flappy Bird - Snake-Tic Tac Toe), whose source code has been Refactored by team members to add the ability to control the settings available in each game by a simple graphical user inter face (GUI) Based on Tkinter and PyQt5.

Technologies and Techniques




Tkinter

    is the only frameworks that’s built into the python standard library, has several strengths It’s cross-platform, so the same code works on Windows, macos, and Linux. Visual elements are rendered using native operating system elements, so applications built with tkinter look like they belong on the platform where they’re run. Although tkinter is considered the de-facto Python GUI framework, it’s not without criticism. One notable criticism is that GUIs built with tkinter look outdated. If you want a shiny, modern interface, then tkinter may not be what you’re looking for. so that we used PyQt5.

PyQt5

    is a GUI widgets toolkit. It is a Python interface for Qt, one of the most powerful, and popular cross-platform GUI library. PyQt was developed by RiverBank Computing Ltd. PyQt API is a set of modules containing a large number of classes and functions. While QtCore module contains non-GUI functionality for working with file and directory etc., QtGui module contains all the graphical controls. In addition, there are modules for working with XML (QtXml), SVG (QtSvg), and SQL (QtSql).

SQLlite3

    is a C-language library that implements a small, fast, self-contained, high-reliability, full-featured, SQL database engine. SQLite is the most used database engine in the world. SQLite is built into all mobile phones and most computers and comes bundled inside countless other applications that people use every day.

GitHub

    is a for-profit company that offers a cloud-based Git repository hosting service. Essentially, it makes it a lot easier for individuals and teams to use Git for version control and collaboration. GitHub’s interface is user-friendly enough so even novice coders can take advantage of Git. Without GitHub, using Git generally requires a bit more technical savvy and use of the command line.

PEP 008

    is a document that provides guidelines and best practices on how to write Python code. It was written in 2001 by Guido van Rossum, Barry Warsaw, and Nick Coghlan. The primary focus of PEP 8 is to improve the readability and consistency of Python code. PEP stands for Python Enhancement Proposal, and there are several of them. A PEP is a document that describes new features proposed for Python and documents aspects of Python, like design and style, for the community.

Privacy Policy

What data do we collect؟

As a user, we collect information that has been mentioned in the part of the databases, which is the first and last name, phone number, email, gender, credit card number and its code, as well as the country and password.

Information usage.

It does not use this data for any profit or other purpose. What you write on your device stays on your device. You can view the code for the project to make sure, which means that we are not responsible for any data leakage that occurs through this application. Security depends on what has been provided in the program. Data is not tampered with in the database, so the security of your personal device or the security of the network connected to it is the responsibility of the user himself and the development team has no responsibility for what happens.

Information security.

All data is stored in the database as plain text, and we put an encryption algorithm that stores the data encrypted inside the database, so no breach process occurs, and all data is leaked, which helps to realize the situation a little faster. The version uploaded to Git Hub contains MD5 Hashing.
